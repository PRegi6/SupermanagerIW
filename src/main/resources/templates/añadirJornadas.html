Añadir jornadas

Sube los con los datos de las jornadas a añadir

Sube los archivos jornada_acb.json y puntos_jugador.json para añadir a la aplicación la información correspondiente.

Añadir estos datos tendrán los siguientes efectos:

  • La jornada actual no se modificará
  • Para cada equipo y cada jugador se añadirán los datos correspondientes a las nuevas jornadas insertadas que no se hayan jugado, es decir si estamos en la jornada 5 solo se insertarán datos de las jornadas mayores que 5
  • Los equipos de los usuarios no se verán afectados de ninguna manera
